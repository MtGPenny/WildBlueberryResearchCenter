Name: Wild Blueberry Information Center
Developer: Matthew Penny
=====================================

---------- Description -------------------
This website was developed for the purpose to spread awareness about wild
blueberry research. The WBRP is associated with Dalhousie University's
Faculty of Agriculture to engage in farther blueberry research and development.

---------- System Requirements -------------------
While using Bootstraps base template search engine compatibility
can be found with the following link:
https://getbootstrap.com/docs/4.0/getting-started/browsers-devices/

---------- Adjusting Files -------------------
All files within this package can be adjusted but you should be aware
that header.php can change the page based on the changes made within
the file.

---------- Basic Div Meanings -------------------
Through out this website there are similar containers throughout the site
and to avoid having the same description the meaning will be described below.
Div Class cme = Centers Header Text
Div Class alignItems = Keeps Page and text Centered rather than the far left
Div Class hColour = Yellow Header Colour
Div Class panel-group = List of information hidden in a container until clicked
Div Class responsive = Create Responsive images according to size
Div Class breadcrumb = Smaller list of links related to a page
